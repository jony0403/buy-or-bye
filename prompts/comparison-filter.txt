역할:
우리가 보고 있는 중고 매물과 검색으로 수집한 후보 매물이 같은 제품인지 판별하세요.
목표는 구매 판단용 가격 참고자료로 쓸 수 있는 후보만 남기는 것입니다. 이 결과를 보장된 시세로 단정하지 않습니다.

처리 기준:
- 같은 브랜드/제품군/모델명/세대/용량/사이즈/에디션이 맞는 후보만 match=true로 두세요.
- 색상이나 구성품만 다르고 핵심 모델이 같으면 포함할 수 있습니다.
- 제목이 너무 일반적이거나 다른 세대/다른 모델/악세서리/부품/케이스/호환품이면 제외하세요.
- 가격이 이상하다는 이유만으로 제외하지 말고, 제품 동일성만 판단하세요.
- 확신이 약하면 제외하세요.

기준 매물:
입력:
제품명: {{productName}}
제목: {{title}}
본문: {{body}}
제품 요약(JSON): {{summaryJson}}

후보 매물(JSON):
{{candidatesJson}}

출력 형식:
반드시 JSON만 출력하세요:
{
  "matches": [
    { "key": "후보 key", "same": true, "reason": "같은 제품으로 본 짧은 근거" }
  ],
  "rejected": [
    { "key": "후보 key", "reason": "제외한 짧은 이유" }
  ]
}
